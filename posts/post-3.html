小红书笔记批量制作工具 - 完全教程

项目背景

做小红书内容创作，最耗时的就是写文案和做封面图。每次都要：

  • 拍产品照片
  • 想吸引人的标题
  • 写种草文案
  • 设计封面图
  • 想标签

这个工具帮你自动化整个流程，节省大量时间！

核心功能

输入：3张产品截图 + 产品标题

输出：3张封面图 + 3份完整文案

技术栈

  • Python - 主要编程语言
  • Gemini 2.5 Flash - 多模态AI，理解图片生成文案
  • Streamlit - Web界面框架
  • Pillow - 图像处理

安装步骤

1. 克隆项目

git clone https://github.com/pinacc640/xiaohongshu-tool.git
cd xiaohongshu-tool

2. 安装依赖

pip install -r requirements.txt

3. 配置API

获取 Gemini API Key

export GEMINI_API_KEY="你的密钥"

4. 准备输入

将3张产品截图放入 input/ 目录

使用方法

命令行方式

python main.py --title "产品名称"

Web界面方式

streamlit run web_app.py

然后在浏览器打开 http://localhost:8501

封面图方案

工具提供多种封面图生成方案，按优先级自动回退：

  1. Gemini 2.5 Flash Image - 最高质量，需付费
  2. OpenAI gpt-image-1 - 高质量，需付费
  3. SiliconFlow FLUX.1-dev - 推荐，新用户免费
  4. Pollinations - 完全免费

推荐：注册 硅基流动，新账号送¥14额度，约700张免费图片

输出示例

运行后在 output/ 目录生成：

output/20260529_143022_产品名/
├── note_1.txt          # 第1篇文案
├── note_2.txt          # 第2篇文案
├── note_3.txt          # 第3篇文案
├── cover_1.png         # 第1张封面
├── cover_2.png         # 第2张封面
├── cover_3.png         # 第3张封面
└── summary.json        # 汇总数据

文案风格

生成的文案特点：

  • 务实不夸张
  • 像朋友推荐一样自然
  • 有具体使用细节
  • 适当口语化

注意事项

  • 需要科学上网访问Google API
  • 注意API调用次数，避免超额
  • 生成内容仅供参考，发布前请审核
  • 封面图生成偶尔会失败，工具会自动重试
