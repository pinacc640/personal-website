# 博客发布指南

## 如何发布新文章

### 方法1：复制模板（推荐）

1. **复制文章模板**
   - 进入 `posts` 文件夹
   - 复制 `post-1.html` 文件
   - 重命名为 `post-3.html`（数字递增）

2. **修改文章内容**
   - 用记事本打开 `post-3.html`
   - 修改以下内容：
     - `<title>` 标签中的标题
     - `<div class="post-date">` 中的日期
     - `<h1 class="post-title">` 中的标题
     - `<div class="post-content">` 中的正文内容

3. **更新博客列表**
   - 打开 `blog.html` 文件
   - 复制一个 `<article class="blog-item">` 区块
   - 修改：
     - 日期
     - 标题
     - 摘要
     - 链接地址（改为 `posts/post-3.html`）
     - 标签

4. **上传更新**
   - 使用Git上传到GitHub
   - Netlify会自动更新网站

### 方法2：创建新文章

1. **创建新的HTML文件**
   - 在 `posts` 文件夹创建新文件
   - 文件名格式：`post-数字.html`

2. **复制粘贴模板代码**
   - 从现有文章复制全部代码
   - 粘贴到新文件
   - 修改内容

3. **更新列表页面**

## 文章内容格式

### 标题
```html
<h2>二级标题</h2>
```

### 段落
```html
<p>这是段落内容</p>
```

### 列表
```html
<ul>
    <li>无序列表项</li>
</ul>

<ol>
    <li>有序列表项</li>
</ol>
```

### 代码
```html
<code>代码片段</code>

<pre><code>
代码块
多行代码
</code></pre>
```

### 链接
```html
<a href="链接地址">链接文字</a>
```

### 图片
```html
<img src="图片路径" alt="图片描述">
```

## 发布流程总结

1. 复制 `posts/post-1.html` → 重命名为 `post-N.html`
2. 修改文章内容
3. 在 `blog.html` 添加文章卡片
4. 上传到GitHub
5. 完成！

## 注意事项

- 文件名要用英文和数字
- 日期格式：2026年5月28日
- 标签要有意义
- 文章要有摘要
