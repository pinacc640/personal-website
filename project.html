Project

小红书笔记批量制作工具

3 张截图 → 3 张封面 + 3 份文案 · Gemini 2.5 Flash · 开源

📝 项目介绍

一个务实的小红书内容批量生成工具。上传 3 张产品截图 + 产品标题，自动输出 3 张封面图 + 3 份完整文案（标题、正文、10个标签）。


✨ 核心功能

• 智能文案生成：使用 Gemini 2.5 Flash，看懂产品截图并写出接地气的种草文案

• 自动封面设计：多 provider 自动回退，确保高质量图片生成

• 批量处理：一次运行生成多篇笔记，提高效率

• Web界面：Streamlit 提供友好的可视化操作界面


🚀 工作原理

输入：3张产品截图 + 产品标题

处理：

　• [Gemini 2.5 Flash] → 分析截图，生成3篇不同角度的文案

　• [图像服务链] → 根据文案标题，生成3张封面图

输出：3张可下载封面图 + 3份可复制文案


🎨 封面图方案

• Gemini 2.5 Flash Image - 高质量（需付费）

• OpenAI gpt-image-1 - 高质量（需付费）

• SiliconFlow FLUX.1-dev - 推荐免费方案（新账号送额度）

• Pollinations - 完全免费


📦 快速开始

1. 克隆项目到本地

2. 安装依赖：pip install -r requirements.txt

3. 配置 Gemini API Key

4. 运行：python main.py --title "产品名称"

5. 或启动Web界面：streamlit run web_app.py


📁 输出结构

每次运行会在 output/ 下创建带时间戳的子目录，包含：

• 3篇文案文件（note_1.txt, note_2.txt, note_3.txt）

• 3张封面图（cover_1.png, cover_2.png, cover_3.png）

• JSON汇总数据
